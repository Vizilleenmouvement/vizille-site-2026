🔐 Le Token GitHub

Le token permet de publier directement sur le site. Il est sauvegardé dans votre navigateur.

  • Collez-le dans le champ 🔐 en haut de page
  • Cliquez sur 👁️ pour le voir/masquer
  • Il est mémorisé automatiquement

⚠️ Ne partagez pas ce token publiquement. Demandez-le à Michel si besoin.
